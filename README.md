# Rocket League 2D Pro Online V4

Diese Version lässt das originale Spiel/Canvas/Physik-Feeling unverändert und verbessert nur die Online-UI und Lobby.

## Neu in V4

- Name muss zuerst eingegeben werden
- einfache Raumliste
- Raum erstellen oder per Code beitreten
- klare Lobby mit Raumcode, Spieler-Liste und Ready-Status
- Host sieht sofort, wer Ready ist und wer nicht
- Startbutton zeigt klar, warum er noch gesperrt ist
- 1v1, 2v2, 3v3, 4v4 Menschen vs Bots
- Gäste simulieren die Physik nicht mehr lokal, sondern bekommen den Host-Snapshot

## Render

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```
