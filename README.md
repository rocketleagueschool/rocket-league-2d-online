# Rocket League 2D Pro Online V7

Online-Version mit Original-Spielgefühl, verbesserter Lobby und V7-Netcode.

## Neu in V7

- Besseres Gefühl für Gäste durch Client-Prediction für das eigene Auto
- Smooth Sync / Interpolation für andere Autos und Ball
- Input wird sofort bei Tastendruck und bis zu 60x pro Sekunde gesendet
- Deutscher Schnellchat im Spiel: erst 1-4 Kategorie, dann 1-4 Nachricht
- `G` öffnet im Spiel ein eigenes Chatfeld
- TAB-Scoreboard bleibt erhalten

## Schnellchat

Während dem Match:

- `1` Information: Ich gehe, Verteidige, Brauche Boost, Pass
- `2` Komplimente: Starker Schuss, Save, Schön gespielt, Danke
- `3` Reaktionen: Okay, Wow, Knapp, Berechnet
- `4` Entschuldigung: Sorry, Mein Fehler, Kein Problem, Nochmal
- `G`: eigene Nachricht schreiben
- `TAB`: Statistik halten

## Render

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```
