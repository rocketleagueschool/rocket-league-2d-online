# Rocket League 2D Pro Online V6

Online-Version mit originaler Pro-Grafik/Physik und verbessertem Online-System.

## Neu in V6

- Name-Gate vor allen Online-Aktionen
- Raumliste + Code-Join + private Räume
- Reconnect-System mit gespeichertem Spieler-Token
- Host-Einstellungen: Modus, Bot-Stärke, Spielzeit, Torlimit, Boost, Replay
- Team-Auswahl: Blau / Orange / Zuschauer
- Ready-System mit Offline-/Ping-Anzeige
- Lobby-Chat + Quick-Chat
- Zuschauer-Modus
- Match-Endscreen
- TAB gedrückt halten: Rocket-League-artige Ingame-Statistik mit Score, Goals, Assists, Saves, Shots und Ping

## Start

```bash
npm install
npm start
```

Render:

```text
Build Command: npm install

```
