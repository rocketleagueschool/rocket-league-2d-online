# Rocket League 2D Pro Online V3

Diese Version stellt die originale Pro-Optik und Smoothness wieder her und ergänzt Online-Multiplayer.

## Neu in V3

- Originales Spielfeld, Canvas-Rendering, Physik und HUD bleiben erhalten.
- Name festlegen, bevor man online spielt.
- Raum erstellen, Raumliste durchsuchen oder per Code beitreten.
- Ready-System: Alle Spieler müssen bereit sein.
- Host wählt den Modus in der Lobby.
- Modi: 1v1 PvP, 2v2/3v3/4v4 Menschen gegen gute Bots.
- Jeder Online-Spieler nutzt WASD/Shift oder Pfeile/Enter auf seinem eigenen PC.

## Render Einstellungen

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```
